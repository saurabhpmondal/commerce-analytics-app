# Marketplace Analytics Platform

Modular analytics platform architecture built for:
- GitHub Pages frontend
- Supabase backend
- Large-scale report systems
- Modular KPI calculations
- Modular reports
- Enterprise analytics scalability

Core Principles:
- Strict modular separation
- Additive-only scalability
- Reusable calculations
- Reusable APIs
- Global state management
- Centralized filters
